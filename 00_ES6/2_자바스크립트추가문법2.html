* 구조분해할당

- 변수 선언 및 초기화 또는 swap 등을 배열 또는 객체 생성 구문으로 간단하게 할 수 있도록 고안된 문법
- 객체 또는 배열의 데이터를 꺼내서 (구조 분해) 변수에 대입 (할당) 하겠다.


* 배열관련함수 (순회함수)

1. forEach : 배열의 모든 인덱스에 대해 반복을 수행해주는 함수, jQuery 의 each 메소드와 유사한 구조
2. map : forEach 함수와 마찬가지로 배열에 대한 반복을 수행하나, 매번 리턴되는 값을 다시 받아서 새로운 배열에 차곡차곡 담아주는 구조
3. reduce : 배열에 대한 반복을 수행할 때, 그 해당 배열의 값을 이용해서 특정값을 도출하거나 재가공할 때 사용 

        [ 표현법 ]

        - 참고) jQuery 의 each 메소드
        배열명.each(function(index, item) {

            반복적으로 실행할 구문;
        });
        > index 에는 해당 배열의 0 ~ 마지막 인덱스가 담겨옴
        > item 에는 index 번째 데이터가 담겨옴

        1. forEach
        배열명.forEach(function(item, index) {

            반복적으로 실행할 구문;
        });
        > jQuery 의 each 메소드와 거의 비슷하나, 매개변수의 순서만 다르다!! 주의!!
        > forEach 는 리턴값이 없는 함수

        2. map
        변수명 = 배열명.map(function(item, index) {

            반복적으로 실행할 구문;

            return 리턴할값;
        });
        > map 은 리턴값이 있는 함수 
        > 최종적으로는 배열로 리턴된다!! 
          (매번 function 마다 리턴되는 값이 새로이 배열에 차곡차곡 쌓인 뒤 그 배열이 최종 리턴)

        3. reduce
        변수명 = 배열명.reduce(function(acc, item, index) {

            반복적으로 실행할 구문;

            return 다음반복회차의acc값;
        });
        > reduce 또한 리턴값이 있는 함수
        > 최종적으로는 단일 값으로 리턴된다!!
          (마지막 반복 회차의 리턴값이 최종 리턴되는것)
