ES6 추가 문법 1

화살표 함수

- Arrow function (Lambda)
- 기존의 익명함수를 더 간결하게 표현하기 위한 문법 

        - 기존의 문법
        변수명 = function(매개변수) {
            실행문
        };

        - 화살표 함수 문법
        변수명 = (매개변수) => {
            실행문
        };

        - 기존의 function 키워드를 생략함, 대신 화살표(=>)를 사용함
        - 매개변수가 1개일 경우 괄호() 생략 가능
        - 실행문이 1줄 return 구문일 경우 return 키워드 생략 가능

백틱 기호

- ` (백틱, backtick)
- 기존의 쌍따옴표 또는 홑따옴표가 아닌 `을 사용해 문자열 표현
( `문자열` ) - 단순 문자열 표현도 가능하지만, 주로 문자열 템플릿을 지정해서 사용
- 자바의 System.out.printf() 와 유사한 기능을 제공
- .html 파일 형식에서만 사용 가능 

        변수명 = `문자열 ${변수명} ${계산식} 문자열`;


전개연산자

- 배열의 요소나 객체의 속성들을 나열해주는 연산자
- 배열이나 객체의 깊은복사본을 만들어서 그 내용물을 나열하는 원리


클래스

- 기존 ES5 문법의 생성자함수를 대체하는 문법
- 자바 문법처럼 클래스를 정의해서 내가 원하는 형태의 객체를 생성 가능 

        - ES5
        function 함수명(매개변수) {
            this.속성명 = 매개변수;
            this.메서드명 = function() {
                실행문;
            }; // 화살표 함수 가능
        }

        - ES6
        class 클래스명 {
            constructor(매개변수) {
                this.속성명 = 매개변수;

                // this.메서드명 = function() {}; // 화살표 함수 가능
            }

            메서드명() {
                실행문;
            }
        }

        실질적으로 기존의 생성자함수와 호출 방법, 사용법은 동일함
        약간의 형태만 차이 있음
