ES6 의 추가적인 문법 1

* 화살표함수

- Arrow function (Lambda)
- 기존의 "익명함수" 를 더 간결하게 표현하기 위한 문법 

        [ 표현법 ]

        - 기존의 문법
        변수명 = function(매개변수) {

            실행할 구문;
        };

        - 화살표함수 문법
        변수명 = (매개변수) => {

            실행할 구문;
        };

        - 기존의 function 키워드를 생략하는 대신, () 와 {} 사이에 => 를 적는다!!

        - 기존의 문법과 비교

                기존의 문법                 |         화살표함수 문법
        ======================================================================
        function() { 실행할 구문 }          |       () => { 실행할 구문 }
        function(a) { 실행할 구문 }         |       (a) => { 실행할 구문 }
                                           |            또는
                                           |        a => { 실행할 구문 }
        function(a, b) { 실행할 구문 }      |       (a, b) => { 실행할 구문 }
        function(a, b) { return a + b }    |       (a, b) => a + b

* 백틱기호

- ` (backtick)
- 기존의 쌍따옴표 또는 홑따옴표가 아닌 `` 을 사용해서 문자열을 표현할 수 있다!! (템플릿 형식으로 지정 가능)
- 단순 문자열 표현도 가능하지만, 주로 "문자열 템플릿" 을 지정해서 유용하게 사용 가능
- 템플릿 형식 지정 시, ${ 변수명 } 또는 ${ 계산식 또는 함수식 } 등으로 문자열을 구성할 수 있다!!
- 자바의 System.out.printf 와 비슷한 역할임!!
- .html 파일 형식에서만 쓸 수 있는 문법이다!! .jsp 파일 형식에서는 쓸 수 없는 문법임!! (EL 구문이랑 문법이 겹침) 

        [ 표현법 ]

        변수명 = `문자열${ 값 }이${ 계산식 }저쩌고..`;


* 전개연산자

- 배열의 요소나 객체의 속성들을 쭉 "나열" 해주는 연산자
- 배열이나 객체의 "깊은복사본" 을 만들어서 그 내용물을 나열해주는 원리
- 전개연산자를 잘 이용하면 깊은복사 뿐만 아니라 여러개의 배열이나 객체를 손쉽게 병합할 수 있게 된다!!


* 클래스

- 기존 ES5 문법의 "생성자함수" 를 대체하는 신문법
- 자바 문법처럼 "클래스" 를 정의해서 내가 원하는 형태의 객체를 생성해낼 수 있다. 

        [ 표현법 ]

        - 기존의 생성자함수 문법
        function 함수명(매개변수) {

            // this 라는 객체가 생성된 꼴 {}

            // 일반 속성을 정의 (자바로 따지면 필드부 역할)
            this.속성명 = 매개변수;

            // 메소드 속성을 정의 (자바로 따지면 메소드부 역할)
            this.메소드속성명 = function() {}; // 화살표함수 형태로도 작성 가능

            // return this; // 묵시적으로 생략
        }
        
        자바로 따지면 클래스와 매개변수생성자의 문법이 결합된 느낌임!!

        - 클래스 문법
        class 클래스명 {

            // 생성자
            // > 자바로 따지면 생성자부 역할, 필드부 역할도 겸함
            //   기존 생성자함수의 문법이 그대로 들어감
            constructor(매개변수) {

                // this 라는 객체가 생성된 꼴 {}

                // 일반 속성을 정의
                this.속성명 = 매개변수;

                // 메소드 속성 정의 - 2
                this.메소드속성명 = function() {}; // 화살표함수도 가능
            }

            // 메소드 속성 정의 - 1
            메소드명() {

                실행할 구문;
            }
        }

        기존의 생성자함수와 호출 방법, 사용법은 똑같다!!
